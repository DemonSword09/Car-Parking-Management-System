# Car-Parking-Management-System

## To configure and setup Server

Use the terminal

```bash
cd server
```

install dependencies - 
```bash
yarn add
```
 
convert typeScript file to javaScript files
```bash
yarn watch 
```

to run server 
```bash
yarn dev
```


## To configure and setup Web
```bash
cd ../web
```

install dependencies -
```bash 
yarn add
```
 
to run server - 
```bash
yarn dev
```


### You'll need PostgreSQL and Redis to run server
